Git时如何解决出现：Commit your changes or stash them before you can merge

出现这个问题的原因是其他人修改了xxx.lua并提交到版本库中去了，而你本地也修改了xxx.lua，这时候你进行git pull操作就好出现冲突了。
解决方法:

1).直接commit -m本地的修改

2).通过stash

git stash
git pull
git stash pop
